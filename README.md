# Data Quality Analyzer

A simple web app that analyzes the quality of any CSV file instantly.

## What it does
- Shows row and column count
- Detects duplicate records
- Identifies missing values with percentages
- Classifies columns as numerical or categorical
- Visualizes missing data with a bar chart
- Displays basic statistics

## Tech Stack
Python, Pandas, Streamlit, Matplotlib

## How to run
```bash
pip install streamlit pandas matplotlib
streamlit run app.py
```
